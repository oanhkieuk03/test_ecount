# Testcases i-Aicon Project

[Tiếng Việt](#dự-án-testcases-iaicon) | [English](#testcases-iaicon-project)

## English

### Overview
Welcome to the `testcases-iaicon` repository. This project is dedicated to testing processes and cases related to i-Aicon.

### Folder Structure
- **`AUTO/`**: Contains automated test scripts and related resources for running tests automatically.
- **`MANUAL/`**: Contains test cases, checklists, and documentation for manual testing procedures.

### Getting Started
1. Clone this repository to your local machine.
2. For automated tests, navigate to the `AUTO` directory and follow the specific setup instructions provided there.
3. For manual testing references, please check the documents inside the `MANUAL` directory.

## Tiếng Việt

### Tổng quan
Chào mừng bạn đến với kho lưu trữ `testcases-iaicon`. Dự án này được dùng để lưu trữ các quy trình và kịch bản kiểm thử liên quan đến hệ thống i-Aicon.

### Cấu trúc Thư mục
- **`AUTO/`**: Chứa các đoạn mã kiểm thử tự động (automated scripts) và tài nguyên liên quan để chạy test tự động.
- **`MANUAL/`**: Chứa các kịch bản kiểm thử (test cases), danh sách kiểm tra (checklists) và tài liệu cho quy trình kiểm thử thủ công.

### Bắt đầu
1. Clone kho lưu trữ này về máy tính cá nhân của bạn.
2. Đối với kiểm thử tự động, vui lòng truy cập vào thư mục `AUTO` và làm theo các hướng dẫn cài đặt cụ thể trong đó.
3. Đối với kiểm thử thủ công, vui lòng tham khảo các tài liệu trong thư mục `MANUAL`.
